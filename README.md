# Image-Cartoonify

You can give a cartoon effect to a photo in Python with simple steps. Python is the pool of libraries. It has numerous libraries for real-world applications. One such library is OpenCV. OpenCV is a cross-platform library used for Computer Vision.
We will be Using Opencv library for image processing , blurring image and enhancing edges.


Steps:
1.Load image.
2.Create edge mask.
3.Reduce the color palette.
4.Combine edge mask with the colored image.
